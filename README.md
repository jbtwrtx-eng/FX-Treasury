# FX Treasury

資金管理システム — React + TypeScript

## セットアップ

Node.js（v18以上）が必要です。

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 画面構成

- **左サイドバー**: 13項目のメニュー
- **ダッシュボード**: 今月利益・年間利益・未処理件数のカード表示
- **その他メニュー**: プレースホルダー画面（今後実装予定）

## 技術スタック

- React 19
- TypeScript
- Vite
